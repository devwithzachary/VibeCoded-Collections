# Collections App

A modern Android application for tracking and managing personal collections (books, trading cards, lego, etc.), completely generated by **Gemini AI**.

## üì± Features

*   **Manage Collections:** Create, rename, and delete collections.
*   **Track Items:** Add items to collections with names, descriptions, and photos.
*   **Have/Want Lists:** Mark items as "Have" (owned) or "Want" (wishlist).
*   **Search & Sort:** Quickly find items with search, and sort alphabetically or filter by status.
*   **Offline Backup:** Export your entire database to a JSON file and import it back to restore or move data.
*   **Modern UI:** Built with Jetpack Compose and Material 3, featuring swipe-to-delete, dark mode support, and smooth animations.
